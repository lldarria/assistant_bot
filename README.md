# assistant_bot

A command-line personal assistant developed in Python for managing contacts, phone numbers, birthdays, and notes. <br>

## Features <br>
Add, edit, and delete contacts <br>
Store multiple phone numbers <br>
Search contacts <br>
Manage birthdays <br>
Display upcoming birthdays <br>
Save and load user data <br>
Interactive command-line interface <br>
Technologies <br>
Python <br>
OOP <br>
Pickle <br>
Poetry/Pipenv <br>
Docker <br>
## Project Structure <br>
The project is built using object-oriented programming principles and includes classes for managing contacts, records, and the address book. <br>
## Environment Management <br>
The application uses Poetry for dependency management and virtual environment creation. <br>
Python version and project dependencies are specified in the pyproject.toml file. <br>
## Docker Support <br>
The application can be executed inside a Docker container. <br>
## Build Docker image <br>
docker build -t personal-assistant. <br>
## Run container <br>
docker run -it personal-assistant. <br>
